==========
User guide
==========

DrawTurksHead can be used on the command line, as a Python package or as a C++ library.

Command-line
============

Basic usage::

    $ python -m DrawTurksHead --leads=3 --bights=7

For details of available parameters::

    $ python -m DrawTurksHead --help
